#### Somme

La somme des matrices est la somme de chaque nombres aux mêmes positions

```
   | a b c |
A= | d e f |
   | g h i |

   | j k l |
B= | m n o |
   | p q r |


        | a+j b+k c+l |
A + B = | d+m e+n f+o |
        | g+p h+q i+r |
```
